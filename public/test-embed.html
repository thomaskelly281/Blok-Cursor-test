This page demonstrates how snippets from the Sitecore Snippet Manager can be embedded into external websites.

Note: This is a test page. In a real implementation, you would replace the snippet IDs with actual IDs from your snippet manager.

Example 1: JavaScript Embed

This snippet is loaded dynamically using JavaScript:

Snippet will appear here
<!-- Add this to your HTML head or before closing body tag --> <script src="http://localhost:3000/api/snippets/1/embed.js"></script> <!-- Add this where you want the snippet to appear --> <div id="sitecore-snippet-1"></div>

Example 2: Iframe Embed

This snippet is embedded using an iframe:

Iframe snippet
<iframe src="http://localhost:3000/api/snippets/2/embed" width="100%" height="auto" frameborder="0"></iframe>

Example 3: Targeted Placement

This snippet is placed in a specific location using data attributes:

Targeted snippet
<!-- Add the script tag --> <script src="http://localhost:3000/api/snippets/3/embed.js"></script> <!-- Add this where you want the snippet to appear --> <div data-sitecore-snippet="3"></div>

Event Handling

You can listen for snippet load events to customize behavior:

document.addEventListener('sitecore-snippet-loaded', (event) => { console.log('Snippet loaded:', event.detail); // Custom logic here });
Important: Make sure your snippet manager is running on localhost:3000 for these examples to work. In production, replace the URLs with your actual snippet manager domain.

Benefits of This Approach

  • No CMS Required: Update content without touching your main website
  • Real-time Updates: Changes appear immediately across all sites
  • Easy Integration: Just add a few lines of code
  • Version Control: Track changes and rollback if needed
  • Performance: Lightweight and fast loading
